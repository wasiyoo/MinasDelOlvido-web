# Minas del Olvido (Web + PWA)

Juego HTML5 exportado con **Godot 4.7**, instalable como **PWA** desde el navegador.

## Jugar online

https://wasiyoo.github.io/MinasDelOlvido-web/

## Instalar (PWA)

1. Abre el enlace en **Chrome**, **Edge** o **Safari** (móvil o PC).
2. Espera a que cargue el juego (primera vez puede tardar ~40 MB).
3. En el navegador:
   - **Chrome/Edge (PC):** icono de instalar en la barra de direcciones, o Menu → Instalar app.
   - **Android Chrome:** Menu → **Instalar app** / **Añadir a la pantalla de inicio**.
   - **iPhone Safari:** Compartir → **Añadir a pantalla de inicio**.

La app se abre a pantalla completa (modo standalone).

## Local

```bash
python -m http.server 8080
```

Nota: la instalación PWA requiere **HTTPS** (GitHub Pages ya lo tiene). En `localhost` también funciona.
